Parking Lot Management System

A simple console-based Python application for managing a parking lot.

Features
Add car (with plate, owner, hours)
Remove car
Show all cars
Show only license plates (using map)
Filter cars over X hours (using filter)
Calculate total parking hours
Show current occupancy
Technologies
Python
Lists & Dictionaries
Functions
map() and filter()
How to run
python main.py
Purpose

This project was created as part of my learning journey in Python (SoftUni).
